✅ 렌더링 순서

![webkit](/resources/webkit.png)

- 동작 순서
  1. 브라우저가 HTML을 차례대로 읽기 시작함
  2. `파싱`
     - `HTML` 파싱
     - `CSS` 파싱
  3. `트리` 생성
     - `DOM` 트리 생성
     - `CSSOM` 트리 생성
  4. `Render 트리`(`DOM tree + CSS의 CSSOM 트리 결합`)를 생성
  5. 생성된 `Render 트리`로 `각 노드의 기하학적 형태를 계산(구성)`한다. (`Reflow(Layout)`)
     - `viewport`에 알맞게
       - viewport: 화면에 표시되는 영역
  6. 화면에 `표시`(`Repaint`)
