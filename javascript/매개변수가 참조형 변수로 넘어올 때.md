✅ 매개변수가 참조형 변수로 넘어올 때
* 해당 참조형 변수의 <b>내용 수정</b>은 가능하지만,
* <b>초기화</b>는 불가능하다.
> 내용 변경(가능)
```javascript
let obj = {name: 'Jackson'};
function changeName(obj) {
  obj.name = 'Lee';
  obj.sex = 'Male';
}
changeName(obj); // {name: 'Lee', sex: 'Male'}로 변경되었다.
```
> 초기화(불가능)
```javascript
let obj = {name: 'Jackson'};
function initObj(obj) {
  obj = {};
}
initObj(obj); // {name: 'Jackson'}, 변경되지 않았다.
```
