✅ 순수함수
* 아래와 같이 매개변수로 객체를 받아도 순수함수일 수 있다.
```javascript
const func = (obj, val) => {
    return obj.number + val;
}

func({number: 3}, 5);
func({number: 3}, 5);
func({number: 3}, 5);
func({number: 3}, 5);
// 동일한 매개변수에 항상 동일한 결과를 리턴하고
// 외부에 영향을 주지도, 받지도 않는다.
```
* <b>중요한 특징 - </b> 순수함수는 <b>시점에 영향을 받지 않는다.</b>
