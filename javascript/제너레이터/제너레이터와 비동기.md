✅ 제너레이터와 비동기

* ** 제너레이터는 `흐름 제어(중단 가능)`가 가능하므로, 
  * `일반 함수`
    ```js
    function foo() {
      const data = apiCall();
      console.log(data); // apiCall()의 작업의 완료를 보장하지 못한다.
    }
    ```
  * ** `제너레이터`
    ```js
    function* bar() {
      const data = yield apiCall();
      console.log(data); // yield로 apiCall()의 작업이 끝날때까지 기다릴 수 있다.

      // apiCall()이 만약 Promise와 같은 특정한 값을 반환한다면??
    }
    ```
