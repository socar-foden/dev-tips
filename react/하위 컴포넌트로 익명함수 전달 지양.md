✅ 하위 컴포넌트로 익명함수 전달 지양
* 익명함수로 하위 컴포넌트로 전달 시, 매번 새로운 함수가 생성되므로 불필요한 렌더링이 발생할 수 있다.
```jsx
const ButtonWrapper = () => {
    const onClickHandle = () => {
        // ...
    }

    return (
        <>
            <ChildComponent propFunc={onClickHandle} />
            {/* <ChildComponent propFunc={() => {onClickHandle();}} /> */}
        </>
    );
}
```
